# 📊 Excel Dashboard Project – Financial & Transaction Analysis

## 📌 Overview

This project is an interactive **Excel dashboard** designed to analyze financial and transaction data in a clear and visually appealing way. It provides insights into revenue, transactions, payment methods, and customer behavior across different dimensions.

---

## 🎯 Objectives

* Monitor overall business performance
* Track revenue trends over time
* Analyze transaction status and success rates
* Understand customer preferences (payment methods & device types)
* Compare revenue distribution across regions

---

## 📊 Dashboard Features

### 🔹 Key Metrics

* **Total Revenue**
* **Total Fees**
* **Total Transactions**
* **Total Cashback**

### 🔹 Visualizations

* 📈 Monthly Revenue Trend (Line Chart)
* 🥧 Transaction Status Distribution (Pie Chart)
* 📊 Transactions by Payment Method (Bar Chart)
* 🍩 Transactions by Device Type (Donut Chart)
* 📉 Revenue by Region (Bar Chart)

### 🔹 Filters (Slicers)

* Month filter (Jan – Dec)
* Payment Method filter (Bank Transfer, Credit Card, etc.)

These slicers allow dynamic interaction with the dashboard for better analysis.

---

## 🛠️ Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers for interactivity
* Data Cleaning & Transformation
* Dashboard Design Principles

---

## 📈 Insights You Can Get

* Revenue trends across months
* Percentage of successful vs failed transactions
* Most commonly used payment methods
* Device usage distribution (Android, iOS, Web)
* Regional revenue comparison

---

## 🚀 How to Use

1. Open the Excel file
2. Use the slicers on the left panel to filter data
3. Explore different charts and KPIs
4. Analyze trends and insights interactively
